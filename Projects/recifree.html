About

My fourth and final project in General Assembly's Software Engineering Immersive course created in a team of three.


This is a full stack CRUD application using React and Django authentication.


Recifree is a recipe finding and posting website using themealdb API to find online recipes


Users can create their own recipes and save recipes found online.
