About

My third project in General Assembly's Software Engineering Immersive course created in a team of three.


This is a single page full stack application using MongoDB, Express, React and Node.js.


Crash Course Collective is a website where students studying software development can view/post projects. Our goal was to create an online portfolio for each user.


Users' pages are populated with both their own projects and the projects they have been tagged in as 'collaborators'.
